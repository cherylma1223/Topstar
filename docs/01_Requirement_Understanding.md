# Topstar AI Coach - 需求理解报告

## 1. 产品定位
Topstar AI Coach 是一款基于 Gemini 多模态能力的垂直领域 H5 应用。它定位为乒乓球爱好者的“随身教练”，提供动作纠正、战术分析及器材建议。

## 2. 业务逻辑拆解
### 2.1 核心服务流
* **当家球星 (主打功能)**: 通用型乒乓球 AI，通过 System Prompt 约束其专业性和简练性格。
* **AI 场外指导**: 侧重视频分析。
* **技术点拨 & 器材咨询**: 针对性子服务。

### 2.2 交互逻辑细节
* **欢迎机制**: 不同的服务进入 ChatScreen 时，会根据 `serviceTitle` 注入不同的初始化消息。
* **多模态能力**: 代码中引入了语音合成 (TTS) 和 视频预览。
* **UI 状态控制**: 
    - 使用 `isTyping` 状态模拟 AI 思考。
    - 使用 `action-card` 触发特定交互（如上传视频）。
    - 视频播放器集成了对 Google Drive 链接的特殊解析逻辑。

## 3. 待确认/风险点
* **API 安全**: 当前 API Key 硬编码在前端或 `.env.local`。
* **视频流处理**: 目前主要靠 `Modality.VIDEO` 调用 Gemini，但前端环境对大体积视频上传和 Token 消耗的控制需要优化。
* **持久化**: 现在的历史记录是 Mock 数据 (id 1-5)，刷新即丢，需要引入后端存储。
